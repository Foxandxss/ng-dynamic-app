DocViewer Home

This project's DocViewer presents several free-form HTML pages with Angular components embedded arbitrarily within those pages.

Each page, located in the /docs folder and linked to on the left, illustrates a particular solution to a particular component presentation problem.

The pages are (mostly) ordered from simple to complex.

The long story attempts to explain the Angular Dynamic Content project in detail.

The embedded component demonstration begins right here, on this page, with the FortuneCookieComponent.
